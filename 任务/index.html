    Append Code”使用说明 -[wzh]

    系统中的每个题目最后都有一个“Append Code”项。有些题目的“Append Code”项不为空。这些题目的“Append Code”项中有 append.c文件或 append.cc文件，点击可以打开。 append.c中的代码会被强制追加到任何以C语言提交的代码末尾， append.cc中的代码会被强制追加到任何以C++提交的代码末尾。 “Append Code”项不为空的题目，用C语言编译并评测的程序是提交的代码和 append.c中代码的合并。C++语言则是评测提交的代码和 append.cc中代码的合并。也就是说，仅需要提交一个程序的前半部分，并且这个程序的后半部分已经给出，固定不变。OJ系统会自动把这两部分合并为一个完整的程序进行评测。含有“Append Code”项的题目，是为了测试编写函数和编写类而准备的。

    以题目“A+B Problem”为例，若append.c中已经给出如下代码： 

    int main() 
{ 
    int a, b; 
    while(scanf("%d%d", &a, &b) != EOF) 
    printf("%d\n", sum(a, b));
    return 0; 
}
    提交的代码是 
    #include <stdio.h>
int sum(int x, int y)
{
    return x + y;
}
    最终评测的代码是这两部分合并为一个完整的程序： 
    #include <stdio.h>
int sum(int x, int y)
{
    return x + y;
} 
int main()
{
    int a, b;
    while(scanf("%d%d", &a, &b) != EOF)
        printf("%d\n", sum(a, b));
    return 0;
}

    包含“Append Code”项的题目的做题步骤

    1. 首先，将 append.c或 append.cc中内容拷贝出来，作为你程序最后一部分，编写这个程序的前半部分，包括头文件的定义
    2. 编写程序时，不要通过改动 append.c或 append.cc中所给的那部分程序代码，使你的程序运行正确。因为你无法修改题目里包含的“Append Code”内容，这样的改动在OJ系统上提交是不会起作用的。你应该尝试读懂“Append Code”中的代码，并完善这个程序的前半部分。
    3. 提交时要注意，不要提交 append.c或 append.cc中的内容，只提交你自己编写的那部分代码；除非你想看到一个程序中有双份“Append Code”代码，这往往会出现main()函数重复定义的编译错误。因为“Append Code”中一般都是程序的 main()部分。

    OnlineJudge系统使用说明-[wzh]

    OnlineJudge系统支持多种程序设计语言，包括：C语言、C++、Java、C#、php、Python、Perl、Pascal、Ruby和unix shell等。可以OnlineJudge系统布置作业让学生在线完成，也可以进行在线命题考试。本系统可供编程语言类课程教学使用，也可用来进行数据结构、算法设计与分析等对程序运行效率有要求课程教学使用，也可用来进行操作系统、编译原理、数据库系统等课程的课程设计。
